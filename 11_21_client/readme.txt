# 현황
- 서버의 저장소에 있는 html 파일을 가져오기 (O)
  * Get 방식(O) Post 방식(X)
- 가져온 파일의 외부 상호작용(a태그, form 등) 구현 (x)
- 코드 내부의 설명 주석(X)
(미구현 내용들 추후 구현 예정)

# 사용법
- 용량 문제로 코드 파일(client.c / cpp)만 포함하였기 때문에 프로젝트를 따로 실행해 주셔야 합니다!
- CLI
  1. 서버를 먼저 실행 한 이후 클라이언트 프로그램을 실행하면 문제없이 작동함

- GUI
  1. 라이브러리 추가
    1) 클라이언트 프로젝트 우클릭
    2) NuGet 패키지 관리 클릭
    3) WebView2 설치

  2. 프로젝트 설정
    1) 클라이언트 프로젝트 우클릭
    2) 속성 클릭
    3) 좌측 메뉴에서 링크 -> 시스템으로 이동
    4) 하위시스템(subsystem)을 창(/SUBSYSTEM:WINDOWS)으로 변경
